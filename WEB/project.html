Управление

Ф - фронтальная грань по часовой, Ф` - фронтальная грань против часовой.

З - задняя грань по часовой, З` - задняя против часовой.

Л - левая грань по часовой, Л` - левая грань против часовой.

П - правая грань по часовой, П` - правая грань против часовой.

В - верхняя грань по часовой, В` - верхняя грань против часовой.

Н - нижняя грань по часовой, Н` - нижняя грань против часовой.

- Повернуть кубик вокруг своей оси.

- Вернуть кубик в собраный вид.

S - 'super'. Кнопка отвечающая за режим прокрутки:
Красная - кубик крутится вокруг своей оси (с помощью мыши, и кнопки);
Синяя - грани крутятся с помощью мыши (но кнопки все еще остаются рабочими).
